# Monkeyin' Around

A physics-driven couch co-op party game where you and your friends swing, fling, and desperately cling to each other to survive a deadly jungle canopy without dropping anyone into the danger below. Use two-button grabbing and momentum to slingshot a chain of panicked primates across hazardous terrain.

## Controls

| Input | Action |
|-------|--------|
| Left click | Grab / release with left hand |
| Right click | Grab / release with right hand |
| WASD | Move |

## Requirements

- [Godot 4.6.1](https://godotengine.org/) with .NET / C# support
- .NET 8 SDK

## Running

Open `project.godot` in Godot 4.6.1 and press F5, or build and run via the .NET CLI with `MonkeyinAround.sln`.
